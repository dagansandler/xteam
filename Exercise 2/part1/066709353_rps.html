Welcome to my rock-paper-scissors JavaScript game!

My impelmentation is as follow:

  • There are 4 functions:
    1. rps(item1, item2) - The main function as describe in the assignment.
    2. generateIfNeeded(item1) - If one of the given arguments are illegal or empty, generate randomly item (rock, paper or scissors).
    3. numberToRPS(number) - Helper method, helps to convert from numbers to rock, paper and scissors.
    4. whoIsTheWinner(item1, item2) - Core method, deside who is the winner as the game describe.

  • The method rps is the main pivot that invoke the 3 other functions.

  • First, there is an input check for the rps parameter with the help of generateIfNeeded method. If needed, generateIfNeeded generate a random item.

  • Atfer that, the winner is detemine with the help of whoIsTheWinner method.
