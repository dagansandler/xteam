Rock, Paper, Scissors

This is a JavaScript version of the game "Rock, Paper, Scissors"

How the JavaScript function that runs the game works:

  • The function takes two arguments as input for player 1 and 2
  • Input can be either 'rock', 'paper', or 'scissors'
  • In case input is missing or illegal, random values are assigned for the player
  • The function then simulates the game between the players
  • The result is then being printed to the DOM
  • To run the game, call the function 'rps' in the console
