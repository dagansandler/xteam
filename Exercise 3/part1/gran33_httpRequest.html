Http Request Parsing

To parse an http request, call "parseRequest" function from the console with an http request string

The function will parse your request as follows:

  1. Check the argument is defined and is a string
  2. Request line parsing
    • Find the request line ending (ends with CRLF)
    • Split the request line by spaces
    • Save the request line method and uri
    • Cut out the resource and the resPath from the URI
  3. Headers Parsing
    • Find the end of the headers (two CRLFs)
    • Split the headers by CRLF
    • For each header, split by ':' and store key and value into the header array
  4. The body is all that's left!

The described parsing returns an empty Object whenever it fails to parse some part of the request.

An example http request:

"POST /xteam/index.html HTTP/1.1\r\nheader1:value1\r\nheader2:value2\r\nheader3:value3\r\n\r\nThis is the request message!"
