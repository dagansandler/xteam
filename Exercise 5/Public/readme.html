What is this app?

This is a basic email app built for the Adavanced Internet Technologies course at BGU, Spring semester 2013
It was built by Ran Greenberg and Dagan Sandler

What it does?

  • This is a realtime email app
  • It allows users to exchange messages
  • Users should register once.
  • After registration they can login and send e-mails
  • If the e-mail has been read, the icon on the right change

How does it work?

  • In order to load the server, mongodb and redis must be running
  • From the server folder, execute 'npm install'
  • After that, simply execute 'node myServer.js' to load the module and start the server
  • Access the app at 'localhost:8080/welcome.html'
  • All client side work is done by Backbone.js
  • Communication is done using Ajax calls, Redis and Socket.io
  • Server side work is done by Node.js and MongoDB
